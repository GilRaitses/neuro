Background

Memory retrieval in the hippocampus is governed by competitive activation of engram populations. Inhibitory circuits regulate this competition by setting activation thresholds and suppressing interfering traces, enabling context-dependent recall without erasing prior memories.

GABAergic interneurons regulate competition between engram populations by setting activation thresholds and suppressing overlapping activity. This inhibitory control determines which neural ensembles are expressed during retrieval. These dynamics are strongly context-dependent and rely on entorhinal input to bias hippocampal activity toward behaviorally relevant memory traces.

At the output level, CA1 encodes memory through temporal ensemble structure rather than firing rate alone. Spike timing relationships, phase coordination and cross-correlated activity patterns define the active memory state during retrieval.

The medial entorhinal cortex provides a critical input to CA1 through the temporoammonic pathway. This input is tightly regulated by local inhibitory interneurons, particularly parvalbumin-expressing cells, which enforce theta phase locking and constrain the timing of excitatory output.

Medial septum GABAergic projections target these interneuron populations and control rhythmic coordination within MEC, thereby shaping the temporal structure of entorhinal output that reaches CA1.

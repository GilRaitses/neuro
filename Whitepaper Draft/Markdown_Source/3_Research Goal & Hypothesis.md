# Research Goal and Hypothesis

The research goal is to explain how inhibitory timing biases selection when overlapping representations compete during retrieval. The relevant system is a memory circuit in which a prior association and an updated association can both remain available. The question is not whether one representation exists. The question is which representation gains control over behavior under a particular state and context.

The hypothesis is that a timing shift in the inhibitory selection system changes which memory pattern stabilizes first. When inhibition narrows or opens temporal windows, one assembly can gain access to the active retrieval pathway while a competing assembly remains present but behaviorally inactive. The selected pattern should then shape perception and certainty until the retrieval state changes, with action following the stabilized pattern.

The hypothesis requires a separation between presence and control. If a competing representation can return under another context, the representation was not erased. If behavior follows only one representation during the retrieval moment, selection has occurred. If state changes alter behavior without changing which representation is active, the effect is not a selection mechanism but a general state effect. A useful account must therefore distinguish timing-based selection from broad arousal, anxiety, or nonspecific activation.

The falsifiable claim is direct: perturbing the inhibitory timing system should shift the dominant retrieval pattern and the behavioral output together. A shift in feeling or movement alone would not be enough. A shift in neural activity without a corresponding behavioral commitment would also be incomplete. The mechanism only holds if timing changes which representation becomes dominant and behavior follows that selected state.

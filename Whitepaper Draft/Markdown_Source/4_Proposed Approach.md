Proposed Approach

This project will synthesize existing circuit literature to test whether medial septum GABAergic input to medial entorhinal cortex can explain retrieval after sequential learning in the same context.

The analysis is organized around three linked circuit steps.

First, the project will examine medial septum GABAergic projections to medial entorhinal cortex as the upstream timing source. This step will focus on evidence that septal GABAergic activity coordinates theta-linked rhythmic structure in MEC and targets local inhibitory populations that regulate circuit timing.

Second, the project will examine MEC inhibitory microcircuits as the transformation layer. Parvalbumin-expressing interneurons will be treated as the main candidate mechanism because they constrain spike timing and enforce theta phase locking of excitatory output. This step isolates timing as the controlled variable, distinguishing temporal coordination from changes in firing rate or overall excitability.

Third, the project will examine CA1 ensemble dynamics as the retrieval readout. Literature on cross-correlation structure, phase coordination and temporal coding will be used to evaluate whether altered entorhinal input timing can reorganize CA1 ensemble composition during retrieval.

The synthesis will evaluate whether the MS GABA to MEC to CA1 pathway provides a sufficient circuit explanation for retrieval selection after sequential learning. Support would require a match across all three levels: septal control of MEC timing, MEC inhibitory control of entorhinal output and CA1 temporal ensemble reorganization during behavior.

Motivation and Gap

Memory updating in the hippocampal–entorhinal system does not eliminate earlier memory traces. Instead, multiple engrams persist and compete during retrieval, with inhibitory circuits regulating their activation thresholds and preventing interference    . While this establishes that retrieval depends on competition rather than erasure, the circuit-level mechanism that selects between competing traces after sequential learning in the same context remains unresolved.

Recent work identifies medial septum GABAergic projections to medial entorhinal cortex as necessary for retrieving updated memory after sequential learning. However, this finding leaves a specific gap. It does not explain how septal GABA input is transformed within MEC into a signal that reorganizes hippocampal output.

The unresolved question is therefore how septal GABAergic input to medial entorhinal cortex alters the timing of entorhinal output such that CA1 expresses one learned value rather than another during retrieval.
